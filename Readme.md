### NODE JS - GPIO

_Le premier qui met un point virgule je le tue_

## Installation

```
npm install
```

## Executer le script

Cela executera Webpack ainsi que node. L'ip est detecter automatiquement et est affiché lorsque le
serveur sera lancé.
```
npm run node
```
